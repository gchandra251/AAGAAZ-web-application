# AAGAAZ Charitable Trust - Web Platform

A professional, responsive web application built for **AAGAAZ Charitable Trust**. This platform is designed to showcase the trust's mission, manage vocational programs, and highlight community impact stories.

## 🚀 Project Goals
* **Dynamic Content:** Moving from static HTML to a Django-powered backend.
* **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
* **Community Engagement:** Sections for "What We Do," "Impact Stories," and "Get Involved."

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables), FontAwesome, Google Fonts.
* **Backend:** Django (Python), SQLite3.
* **Tools:** Git/GitHub for version control.

## 📈 Status: In Progress
Currently implementing the Django Admin backend to allow non-technical staff to update program details and contact information dynamically.
